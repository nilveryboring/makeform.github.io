# How to create a free quiz with Makeform

---

## Create a quiz

With Makeform, you can build and personalize your own quiz in minutes.

<div className="shadow-2xl mt-12">
  ![Free Quiz Example1](/images/free-quiz1.png)
</div>

You can click the tags below the chatbox to start the prompt quickly, then specify the number of questions,
the type of questions, and the options.
