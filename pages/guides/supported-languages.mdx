## How to Change Language

The language setting is automatically detected from your browser preferences. To use Makeform in a different language:

### 1. Using the Language Dropdown

Click the language dropdown at the bottom of the left sidebar (next to the theme switcher) to select your preferred language.

![Language dropdown in sidebar](/images/supported_languages1.png)

### 2. Access via URL

Add the language code to the URL:
- Example: `makeform.ai/fr` for French
- Example: `makeform.ai/ja` for Japanese
- English doesn't require a language prefix

**Language codes:**
- `de` - German
- `ja` - Japanese
- `es` - Spanish
- `fr` - French
- `ko` - Korean
- `it` - Italian
- `th` - Thai
- `ru` - Russian
- `zh-Hant` - Traditional Chinese
- `pt-BR` - Brazilian Portuguese
- `ar` - Arabic

<Callout type="info">
Once you change the language, the interface will remain in that language for your session.
</Callout>

---

## What Gets Translated

When you use Makeform in a supported language, the following elements are automatically translated:

- Dashboard and navigation menus
- Form builder interface
- Settings and configuration options
- Default form text and labels
- Button text and actions
- Error messages and notifications
- Help text and tooltips

<Callout type="info">
Your form content (questions, answers, custom text) is not automatically translated. You control the language of your form content.
</Callout>
