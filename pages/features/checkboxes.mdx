# Checkboxes

The Check Box component allows users to select multiple options from a predefined list. Checkboxes enable the selection of multiple values simultaneously, making them useful for collecting preferences, multiple-choice answers, or consent confirmations.

---

- **Multiple Selections ** – Users can check one or more options.
- **“Other” Option** – Allows users to specify a custom response.
- **Flexible Customization** – Adjust label text, default selections, and styles.
