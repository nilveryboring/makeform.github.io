# Multi Select

The Multi Select component allows users to select multiple options from a dropdown list. Selected options appear inside the input field, and users can remove them individually if needed. This component is useful when users need to pick multiple values from a predefined list.

---

- **Select Multiple Values** – Users can choose more than one option.
- **Removable Selections** – Each selected item can be removed individually.
- **Real-Time Selection Updates** – Selected values appear dynamically in the input field.
- **Search & Filtering** – Users can quickly find options from a long list.
