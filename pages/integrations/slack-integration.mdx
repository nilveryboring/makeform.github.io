# Slack Integration

Send new Makeform submissions to a Slack channel so your team can review responses without opening the dashboard.

<Callout type="info">
Slack automations are created from a form's Automations tab. You can connect a Slack workspace once, then use it across forms.
</Callout>

## What the Slack integration does

When someone submits your form, Makeform can post a message to a selected Slack channel.

Use it for:

- Lead alerts
- Support intake
- Application review
- Internal approvals
- Payment or order notifications
- Team visibility on high-value forms

## Connect Slack

1. Open the form you want to automate.
2. Go to the <span className="text-cyan-500 font-semibold bg-cyan-100 px-2 py-1 rounded">Automations</span> tab.
3. Choose <span className="text-cyan-500 font-semibold bg-cyan-100 px-2 py-1 rounded">Slack</span>.
4. Connect your Slack workspace if it is not connected yet.
5. Select the channel where Makeform should post.
6. Save the automation.

After setup, submit a test response and confirm the message appears in Slack.

## Use multiple Slack automations

You can create more than one automation when different forms or workflows need different destinations.

For example:

- Send sales leads to `#sales`.
- Send bug reports to `#support`.
- Send job applications to `#hiring`.

## Testing checklist

- Confirm the Slack workspace is connected.
- Choose the correct channel.
- Submit a test response.
- Check that Slack receives the message.
- Disable or remove old Slack automations that should no longer run.

## Troubleshooting

### Slack does not receive a message

Check that the automation is enabled and the selected Slack channel still exists.

### The wrong channel receives messages

Edit the Slack automation and choose the correct channel.

### The Slack connection stopped working

Reconnect the Slack workspace, then test the automation again.
